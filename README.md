# Gulp + Tailwind Starter

Шаблон для быстрого старта проектов с Gulp и TailwindCSS.

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/ваш-username/gulp-pug-tailwind-template.git новый-проект

# Перейти в папку проекта
cd новый-проект

# Удалить историю git (начать свою)
rm -rf .git

# Инициализировать новый git репозиторий
git init

# Установить зависимости
npm install

# Запустить разработку
npm run dev
